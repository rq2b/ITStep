```python
# Вирівнювання тексту:
# < (left align)
# > (right align)

print(f'|{'Текст':<10}  |')
# Результат: |Текст       |

print(f'|{'Текст':>10}  |')
# Результат: |     Текст  |

print(f"| {'Маська':<10} | {'760':>6} грн. |")
print(f"| {'Буся':<10} | {'8500':>6} грн. |")
# Результат:
# | Маська     |    760 грн. |
# | Буся       |   8500 грн. |

'''
- `{:<N}` — ліворуч у полі шириною N символів
- `{:>N}` — праворуч у полі шириною N символів
- Зручно для вирівнювання колонок / значень
'''
```
